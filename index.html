About Me

I am a proud father to twin girls, Talia and Tiara. I've worked in many different fields, yet never felt like those were my calling until I stumbled upon Web Development. I was tasked to build a site for a business platform my employer was unveiling to new clients. I researched quite a few solutions for this request and found that wordpress and wix did not allow me the creative freedom that I wanted and the user interface was quite tedious. I decided it was time to learn web development in order to create a better opportunity for myself to further advance in my career. By divine intervention I searched for web development courses and found the UM Coding BootCamp program.
